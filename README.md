## 📐 Как работает алгоритм

Алгоритм Борувки (1926) строит MST за **O(log V) фаз**, на каждой из которых число компонент уменьшается минимум вдвое:

```
1. Инициализация — каждая вершина является отдельной компонентой
2. Повторять, пока компонент > 1:
   а) Каждая компонента выбирает минимальное исходящее ребро
   б) Все выбранные рёбра добавляются в MST
   в) Соответствующие компоненты объединяются
3. Результат — минимальное остовное дерево (или лес)
```

**Временная сложность:** `O(E · log V)`  
**Пространственная сложность:** `O(V + E)`

Структура данных DSU (система непересекающихся множеств) использует оптимизации **сжатия пути** и **рангового объединения**, обеспечивая амортизированную сложность операций `O(α(V))`.

## 🌐 API

| Метод | Маршрут | Описание |
|---|---|---|
| `GET` | `/` | Главная страница |
| `POST` | `/run` | Запуск алгоритма (JSON: `{n, edges}`) |
| `GET` | `/example/<id>` | Получить готовый пример графа |
| `GET` | `/examples` | Список всех примеров |
