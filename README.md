# Алгоритм Борувки — Веб-приложение

> Веб-приложение для интерактивной демонстрации работы алгоритма Борувки построения минимального остовного дерева (MST).

## 🚀 Возможности

- 🔹 **Построение MST** — нахождение минимального остовного дерева для связного взвешенного графа
- 🔹 **Поддержка несвязных графов** — построение минимального остовного леса
- 🔹 **Пошаговая трассировка** — визуализация каждой итерационной фазы алгоритма
- 🔹 **Готовые примеры** — 5 предустановленных тестовых графов
- 🔹 **Валидация ввода** — проверка корректности данных перед запуском
- 🔹 **Адаптивный интерфейс** — работает на десктопах и мобильных устройствах

---

## 🛠 Стек технологий

- Python 3.8+
- Flask (микрофреймворк)
- HTML / CSS / JavaScript (нативный, без зависимостей)
- Структура данных DSU с оптимизациями (сжатие пути + ранговое объединение)

## ▶️ Запуск

# 1. Клонировать репозиторий
git clone https://github.com/your-username/boruvka-app.git
cd boruvka-app

# 2. Установить зависимости
pip install flask

# 3. Запустить приложение
python app.py
🌐 Приложение откроется по адресу: **http://127.0.0.1:5000**
