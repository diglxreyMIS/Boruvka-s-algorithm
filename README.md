Алгоритм Борувки — Веб-приложение
Веб-приложение для интерактивной демонстрации работы алгоритма Борувки построения минимального остовного дерева (MST — Minimum Spanning Tree).
Курсовая работа по дисциплине «Алгоритмы и структуры данных», НИТУ МИСИС, группа БИВТ-24-2.
Возможности
🔹 Построение MST — нахождение минимального остовного дерева для произвольного связного взвешенного графа
🔹 Поддержка несвязных графов — корректное построение минимального остовного леса для графов с несколькими компонентами связности
🔹 Пошаговая трассировка — детальная визуализация каждой итерационной фазы: выбор дешёвых рёбер, объединение компонент, состояние DSU
🔹 Готовые примеры — 5 предустановленных тестовых графов для мгновенного запуска демонстрации
🔹 Валидация ввода — проверка корректности данных: диапазоны вершин, отсутствие петель, положительные веса
🔹 Адаптивный интерфейс — удобное отображение на десктопных и мобильных устройствах
